== Description ==

Advance Category Posts Widget provides a smart widget that shows posts from the selected category using tons of options. The plugin provides a handy widget 'Advance Category Posts Widget'. 

This widget shows posts from the selected category in 3 major options:

1. Slider:  The posts from the selected category is shown in the slider. And this option provides ability to control slider navigation, slide direction and slider pagination.

2. List: The posts from the selected category will be shown in the list. The post will be listed along with the post thumbnail (featured image), post title and the category name.

3. Time: This option generally comes handy for the magazine sites. The post will be changed according to the number or hours or days. New post will appear after every selected duration.

== Image size ==

Use `acpw_image_size` filter to overwrite the image size.  

== Installation ==

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select advance-posts-categroy-show.zip from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download advance-posts-categroy-show.zip
2. Extract the advance-posts-categroy-show directory to your computer
3. Upload the advance-posts-categroy-show directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard
